# Дифференциатор

## Описание
Эта программа предназначена для аналитического вычисления производной заданного арифметического выражения относительно переменной `x`. Поддерживаются основные математические операции и элементарные функции.

## Возможности
- Дифференцирование многочленов (например, `x^2 + 3*x + 5`)
- Поддержка тригонометрических функций (`sin(x)`, `cos(x)`, `tan(x)`, `ctan(x)`)
- Обработка экспоненциальных и логарифмических функций (`e^x`, `log(x)`)
- Учёт правил дифференцирования (правило суммы, произведения, частного, цепное правило)
- Упрощение полученных производных
- Вычисление производных второго и более порядка
- Отображение результата в формате pdf

## Установка
1. Клонируйте репозиторий
2. Выполните команду `make`

## Использование
- Файл с исходным выражением поместите в папку `data`.
- Для использования запустите исполняемый файл `main`, передав в качестве аргументов имя файла и порядок производной
- Для отображения в pdf вызовите `make tex`

## Ограничения
- Не поддерживается неявное умножение (нельзя писать `2x` вместо `2*x`)
- Может потребоваться ручное упрощение сложных выражений
